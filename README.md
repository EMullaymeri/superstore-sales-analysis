# Superstore Sales Analysis

## Overview
Analysis of 51,290 rows of global superstore sales data (2011-2014).

## Tools Used
- Google Sheets — data cleaning and pivot table analysis
- MySQL — SQL queries for data analysis
- Canva — presentation and data visualization

## Key Findings
- Sales nearly doubled from 2011 to 2014
- Profit margins declining from 9.1% to 8.5%
- Tables is the only loss-making sub-category at -8.47%
- Southeast Asia has 8 unprofitable sub-categories driven by 27% average discount

## Recommendations
- Reprice or discontinue Tables
- Review discount policy in Southeast Asia
- Invest more in Copiers and Accessories
